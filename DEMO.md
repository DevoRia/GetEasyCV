## üöÄ Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Open Browser**:
   Navigate to `http://localhost:3000`

4. **Start Editing**:
   - Modify the template in the left editor
   - See live preview on the right
   - Export to PDF when ready
